BlockstackAdvancedExample

This repository uses a Clean Architecture with an Application layer that uses MVVM, RxSwift and Coordinators. (Factories are being used to reuse viewcontrollers accross coordinators.) The Blockstack SDK is seperated into a Platform layer to only import Blockstack in one place. In this app the Domain layer is renamed to Core.
